go-mysql-react
==============

Go-restful working with react, mysql + bcrypt, and bootstrap

	go get "code.google.com/p/go.crypto/bcrypt"
	go get "github.com/emicklei/go-restful"
	go get "github.com/go-sql-driver/mysql"

MySQL Users Table:

	id: int(16)
	name: varchar(128)
	last: varchar(128)
	email: varchar(128)
	password: varchar(255)
